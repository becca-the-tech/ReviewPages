Tips I Learn

  • ! + tab on an empty HTML document will set up the boilerplate for the document
  • HTML_EL * num and then a tab will multiply the HTML_EL by number (li * 5 will create 5 list elements upon pressing tab)
  • h2 > ul > li will create an h2 with a ul inside it, and the ul will have an li
  • table elements: colspan and rowspan are available attributes
