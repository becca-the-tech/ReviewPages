MongoDB notes

  • show dbs -> lists existing databases use certainDB -> changes the db in use to certainDB show collections -> shows existing collections within the DB
  • db.collectionName.insertOne({name: valueName, age: valueAge}) -> inserts a single JS object into the collectionName with the key-value pairs listed in the parentheses
  • db.collectionName.find() will show you which documents (the objects you've added to the collection) are part of that collectionName
  • db.collectionName.insertMany([{ name: "object1", type: "whatever"}, {name: "object2", type: "something"]) -> takes in an array of objects and adds them as documents to the collectionName
  • db.collectionName.insert() -> can take either a single object or an array of objects
  • db.collectionName.find({name: "object1"}) -> find can take in an object just containing the key-value options you want to find, but it is case-sensitive
  • db.collectionName.findOne({isBoolean: true}) -> will find the first matching result, returns the actual document whereas find() returns a cursor (which is like a pointer to the documents)
  • db.collectionName.updateOne({name: "object1"}, {$set: { type: "what-whatever"} } ) -> the way of calling updateOne is the first object is what you are selecting based on, then the second object is {$set: {} } with the innermost brackets being which fields you are updating
  • db.collectionName.updateOne({name: "object1"}, {$set: {type: "umm whatever"}, $currentDate: {lastChanged: true} }) -> will update the document that has a name of "object1" and change its type to "umm whatever", but it will also add a lastChanged field of type ISO-date set for the time when updated
  • db.collectionName.deleteOne({filter: true}) -> will delete first document it finds that matches the selector condition
  • db.collectionName.deleteMany({filter: true}) -> will delete all documents that match true for the filter condition, leaving it blank will delete everything from a collection
